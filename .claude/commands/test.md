Run `make test` to execute the unit test suite and report results.

Follow these steps:

1. Run the test command:
   ```
   make test
   ```

2. If any tests fail:
   - Analyze the failure output to identify root causes
   - Fix the failing tests or the code causing failures
   - Re-run `make test` to verify all tests pass

3. Show a summary of the test results.
