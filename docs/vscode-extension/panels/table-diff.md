# Table Diff Panel

The Table Diff panel lets you compare two tables and see how they differ, without leaving VS Code. It's located at the bottom of the interface, near the `Lineage` and `Query Preview` panels.

## Functionality

- **Compare two tables**: Pick two tables from your connected databases and compare them side by side.
- **Schema differences**: See which columns and types were added, removed, or changed between the two tables.
- **Data differences**: Compare row counts and column-level statistics to spot data drift or mismatches.

This is useful for checking a model against its source, or for confirming that a change produced the data you expected.

## How to Use

1. Open the Table Diff panel from the bottom panel area.
2. Select the two tables you want to compare.
3. Review the schema and data differences shown in the panel.
