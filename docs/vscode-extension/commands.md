# Commands & Shortcuts

You can run most Bruin actions from the buttons in the UI, the Command Palette (`Ctrl/Cmd + Shift + P`), or a keyboard shortcut.

## Commands

| Command | What it does |
|---|---|
| **Bruin: Render** | Render the current asset's SQL (Jinja and materialization applied). |
| **Bruin: Install Bruin CLI** | Install the Bruin CLI from within VS Code. |
| **Bruin: Convert File to Asset** | Turn the current `.sql`, `.py`, or `.yml` file into a Bruin asset. |
| **Bruin: Preview Selected Query** | Run the selected SQL and show the result in the Query Preview panel. |
| **Bruin: Toggle Folding** | Fold or unfold the Bruin sections in the file. |
| **Bruin: Refresh Connections** | Refresh the connections shown in the Databases view. |
| **Bruin: Show Getting Started Walkthrough** | Open the walkthrough that introduces the extension's main features. |

From the **Databases** view you can also refresh an individual connection or schema and open a table's details from its right-click menu.

## Keyboard shortcuts

| Action | Windows/Linux | Mac |
|---|---|---|
| Preview Selected Query | `Ctrl+Shift+R` | `Cmd+Shift+R` |
